# ddd-learning

エリック・エヴァンスの『ドメイン駆動設計』を読みながら、
投資判断補助ツールの実装を通じてDDDを学んだ記録です。

## このリポジトリについて

各章のサンプルコードと、対応するZenn記事をセットで公開しています。
学習中の記録なので、随時更新していきます。
全部書ききれるかは未定で、その時の関心に応じて進めます。

## 統合実装

各章で扱った概念を、投資判断補助ツールという題材で統合的に実装しています。
詳細は [investment-tool/README.md](./investment-tool/README.md) を参照。

このツールが目指すスコープ：
- 個人投資家のポートフォリオ管理
- 取引履歴の記録
- 評価額計算とアラート

意図的に範囲外としたもの：
- 証券会社APIとの連携（モックで代用）
- 税務計算
- 機械学習モデルの自前実装
